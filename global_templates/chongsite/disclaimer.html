About this Site

This site is powered by Django, which is a full stack web framework written in Python. The html css3 design is borrowed from online. Personally, I think it saved a huge amount of dev time, since I am really not a frontend person.

The site was developed in Django 1.5 on Python 3.3, tied with PostgreSql.

All the content within this site only represents my personal opinion, and they might be wrong. Please let me know if you find anything annoy you or any suggestions are highly welcomed.

This site is still under development. I will keep adding features and cool things, as well as maintain it.
